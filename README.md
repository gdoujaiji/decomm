# decomm
Blockchain Ecommrce

Setup:
1. Smart contract:
    - install Node Js
    - npm install -g truffle
    - truffle init
    - npm init -y
    - npm install @openzeppelin/contracts

2. Backend:
    - npm init -y
    - npm install koa @koa/router @koa/cors ethers mongoose

3. Frontend:
    - npm install -g create-react-app
    - npm install bootstrap ethers axios
